# Data Flow Documentation

## Data Entry Points
1. **Daily Records** - Main entry point for student progress
2. **Student Registration** - New student enrollment
3. **Teacher Assignment** - Updating student's teacher

## Data Processing
1. **Attendance Reports** → Monthly attendance percentages
2. **Progress Tracking** → Student advancement through surahs

## Data Export
- Monthly reports
- Teacher performance summaries
- Student progress certificates

## Data Retention
- Active students: Full history kept
- Inactive students: Archived after 1 year
- Daily logs: Summarized monthly after 2 years
