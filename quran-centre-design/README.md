# Quran Centre Management System - Database Design

## Overview
Database schema for tracking student progress in Quran memorization center.

## Key Features
- Student progress tracking (daily/monthly)
- Multiple teachers and classes
- Hijri calendar support
- Attendance monitoring

## Schema Version: 1.0.0
Last Updated: March 2026

## Quick Start
1. Run `schema.sql` to create tables
2. Run `data/quran_surahs.csv` import for reference data
3. Run `data/sample_data.sql` for test records
